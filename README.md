# 파이썬 기초 강의 

코딩을 처음 배우는 사람들을 위해, 단기간 안에 파이썬 기초 문법과 간단한 응용 프로젝트를 진행할 수 있도록 구성된 강의입니다. 
수업은 주 1회, 1회 2시간 30분 ~ 3시간 이내로 진행되며, 강의안의 틀은 ["점프투파이썬"](https://wikidocs.net/book/1) 을 참고하여 제작되었습니다.
